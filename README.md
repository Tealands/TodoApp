## ローカルで動かす方法

### 必要な開発環境

- windows環境
- Python 3.8以上
- Flask
- その他、`requirements.txt`に記載されているライブラリ

### 手順

1. このレポジトリをクローンする

```bash
git clone https://github.com/Tealands/TodoApp
```

2. クローンしたディレクトリに移動する
3. 必要なライブラリをインストールして、アプリを起動する

```bash
pip install -r requirements.txt
python app.py
```

4. ブラウザで `http://localhost:5000` にアクセスする
